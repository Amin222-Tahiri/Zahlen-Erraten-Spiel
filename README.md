Zahlen-Erraten-Spiel

Ein interaktives Konsolen-Spiel, bei dem der Spieler eine vom Computer generierte Zufallszahl erraten muss. Ein tolles Projekt, um die Grundlagen der Programmierlogik zu demonstrieren.

Beschreibung
In diesem Spiel wählt der Computer eine zufällige Zahl in einem bestimmten Bereich (z. B. 1 bis 100). Der Spieler gibt Tipps ab, und das Programm hilft mit Hinweisen wie "Zu hoch!" oder "Zu niedrig!", bis die richtige Zahl gefunden wurde.

Features
- **Zufallslogik:** Jedes Spiel ist anders durch dynamisch generierte Zahlen.
- **Benutzer-Feedback:** Sofortige Rückmeldung nach jeder Eingabe.
- **Versuchs-Tracker:** Zählt mit, wie viele Versuche benötigt wurden.
- **Fehler-Handling:** Das Programm stürzt nicht ab, wenn man versehentlich Buchstaben statt Zahlen eingibt.

Technologien & Konzepte
- **Sprache:** [Z.B. Java / Python / JavaScript]
- **Konzepte:** Schleifen (While-Loops), Bedingungen (If-Else), Zufallsgeneratoren, Benutzereingaben (Input-Handling).
